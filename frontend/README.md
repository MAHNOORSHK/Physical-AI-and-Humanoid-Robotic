# Frontend - Physical AI Textbook

Docusaurus-based interactive textbook.

## Setup

```bash
npm install
npm start
```

## Structure

```
frontend/
├── docs/              # Markdown content files
├── src/               # React components
├── static/            # Images, assets
├── docusaurus.config.js
└── package.json
```

## Adding Content

Create markdown files in `docs/` folder organized by modules.
